# DualSense Controller Support for Cyberpunk 2077

This mod adds Adaptive Triggers support and some other features for the PS5 DualSense controller.

**Half of PS5 Next-gen on PC.**

This mod includes:
1. Over **_50_** manually configured trigger effects for weapons and vehicles.
2. Trigger effects for turrets and brandances.
3. Most of the different events and actions of the player affect the lightbar of the touchpad.
4. Shows the charge level of your controller right in the game. (only in paid DSX)

## Before installing...

You need the accessory program **DualSenseX**.
- If you have a couple of eddies and you are ready to buy the paid version (or you already have it), with all the features, updates and improvements, then you need to go [here](https://store.steampowered.com/app/1812620/DSX/) - Recommended
- If you don't want to pay - go [here](https://github.com/Paliverse/DualSenseX/releases/tag/1.4.9), but keep in mind that some cool features will not be available to you.

## Notes
If controller vibration is not working for you, try disabling all your controller emulation tools, and even the tool that is enabled through the Steam Big Picture. After that, run DualSenseX and select Xbox 360 emulated contoller.

If you don't like Xbox360 prompts but want to play with vibration, install this mod: [Playstation Prompt cyberfix](https://www.nexusmods.com/cyberpunk2077/mods/463?tab=description)

Please report any bugs and flaws, I will try to fix them as soon as possible.
